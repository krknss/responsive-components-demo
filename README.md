# responsive-components-demo – html & css

Selbstständig umgesetztes Frontend-Projekt mit Fokus auf
responsive Layout und strukturierte Komponenten.

## Technologien
- HTML5
- CSS3
- Flexbox
- Media Queries

## Ziel
Entwicklung eines responsiven Komponenten-Layouts
mit sauberer Struktur und wiederverwendbaren Elementen.
